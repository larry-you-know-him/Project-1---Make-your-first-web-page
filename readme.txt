TITLE:		Movie Trailer Website

PURPOSE: 	Show movie art/posters and allow users to play the trailer for a given movie

DATE: 		3/22/2015

HOW TO START: 	Open a web browser then use it to open the file fresh_tomatoes.html to display the movie trailer web page

AUTHOR: 	Lawrence Mello

USER INSTRUCTIONS:
Once the page displays, you may be promted to enable ActiveX content.  Do this to enable the movie trailers to play.
Click on a movie and its trailer will play in a popup box.  You can stop playing the trailer at any time by closing
the popup box.
